Redis是什么

在使用中，我误以为这是个类似插件或者Navicate的软件，不需要做什么就可以使用。所以在是什么这个问题上盘旋了一番。

  • 概念：Redis 是完全开源的，遵守 BSD 协议，是一个高性能的 key-value 数据库，属于NoSQL数据库之一。
  • 特点：基于内存运行，可以通过磁盘实现数据的持久存储； 支持丰富的数据类型（String、list 、hash 、set 、zset）；支持主从同步，即 master-slave 主从复制模式。
  • 优势：Redis 基于内存实现数据存储，它的读取速度较快等
  • 应用场景：缓存一些经常被访问的热点数据、或者需要耗费大量资源的内容，通过把这些内容放到 Redis 中，可以让应用程序快速地读取它们。例如，网站的首页需要经常被访问，并且在创建首页的过程中会消耗的较多的资源，此时就可以使用 Redis 将整个首页缓存起来，从而降低网站的压力，减少页面访问的延迟时间。

Redis的下载安装

以该链接https://www.runoob.com/redis/redis-install.html的内容为基准，在Github上下载64位压缩包，解压后按照教程安装。

打开一个 cmd窗口 使用 cd 命令切换目录到 C:\redis运行:(如把redis路径加入环境变量，省略切换目录)

原窗口不关闭，另启动一个cmd窗口，切换到redis目录下访问服务端，设置键值对并取出该键值对。

RedisInsight的下载安装

RedisInsight是Redis官方出品的可视化管理工具，可用于设计、开发、优化你的Redis应用。支持深色和浅色两种主题，界面非常炫酷。可支持String、Hash、Set、List、JSON等多种数据类型的管理，同时支持远程使用CLI 功能，功能非常强大。使用教程如https://www.cnblogs.com/rickie/p/16634697.html。

  • 安装完成后在主界面选择添加Redis数据库，选择手动添加数据库，输入Redis服务连接信息即可。

  • 打开连接后即可管理Redis，左侧显示数据库的key列表，右侧显示选择key的对应value 值。
