局部变量

在方法体内声明的变量被称为局部变量，在栈上分配。

public class LocalVariable {
    public static void main(String[] args) {
        //a b c为局部变量
        int a = 10;
        int b = 10;
        int c = a + b;
        System.out.println(c);
    }
}

成员变量

在类内部但在方法体外声明的变量称为成员变量，或者实例变量，或者字段。之所以称为实例变量，是因为该变量只能通过类的实例（对象）来访问。

public class InstanceVariable {
    //成员变量
    int data = 88;
    public static void main(String[] args) {
        InstanceVariable iv = new InstanceVariable();
        System.out.println(iv.data); // 88
    }
}

静态变量

通过 static 关键字声明的变量被称为静态变量（类变量），它可以直接被类访问。

public class Example {
    // 静态变量
    static int count;

    public static void main(String[] args) {
        // 可以通过类名直接访问静态变量
        Example.count = 5;

        // 创建多个对象，共享同一个静态变量
        Example obj1 = new Example();
        Example obj2 = new Example();

        // 通过对象访问静态变量
        System.out.println(obj1.count);  // 输出：5
        System.out.println(obj2.count);  // 输出：5

        // 修改静态变量的值
        obj1.count = 10;

        // 所有对象共享修改后的静态变量值
        System.out.println(obj1.count);  // 输出：10
        System.out.println(obj2.count);  // 输出：10
    }
}

count是一个静态变量。我们可以通过类名Example.count或对象名obj1.count、obj2.count来访问静态变量。在创建多个Example对象后，它们共享相同的静态变量，并且修改其中一个对象的静态变量会影响到其他对象。在这段代码中，无需通过创建实例来访问变量，可以直接通过类去访问。

静态变量通常用于在类的所有实例之间共享数据或记录与类相关的信息。它们在许多场景中非常有用，例如计数器、全局配置、共享资源等。需要注意的是，在使用静态变量时应考虑线程安全性和适当的访问控制。

常量

在 Java 中，有些数据的值是不会发生改变的，这些数据被叫做常量——使用 final 关键字修饰的成员变量。常量的值一旦给定就无法改变！

public class FinalVariable {
    final String CHEN = "沉";
    static final String MO = "默";
    public static void main(String[] args) {
        FinalVariable fv = new FinalVariable();
        System.out.println(fv.CHEN);
        System.out.println(MO);
    }
}
